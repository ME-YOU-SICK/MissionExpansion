# Mission Expansion

Mission Expansion is a premium, minimalist learning web app prototype implementing the idea in `App Idea`.

## Features implemented
- Bottom-up lesson flow (vocabulary → concept → checkpoint quiz)
- Mistakes backlog and retention pressure (fading node timer)
- Interactive knowledge web visualization with progress saturation
- “Test-like” checkpoint behavior with score-based progression changes
- Spaced-repetition scheduling utility + automated tests

## Run locally
```bash
npm run dev
```
Open: `http://localhost:4173`

## Run tests
```bash
npm test
```
